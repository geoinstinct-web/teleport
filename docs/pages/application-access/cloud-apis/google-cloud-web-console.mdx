title: "Protect Google Cloud Web Console Access with Teleport"
description: How to enable secure access to Google Cloud Web Console and APIs with Workforce Idenity Federation.
---

You can manage access to GCP web console by integration GCP Workforce Identity Federation
with Teleport SAML identity provider.

Please refer to the GCP [Workforce Identity Federation with Teleport SAML IdP](../../access-controls/idps/saml-gcp-workforce-identity-federation.mdx) guide to learn
more about how the integration works and how to configure one.
