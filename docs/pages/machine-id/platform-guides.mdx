These guides cover the installing and configuring Machine ID on specific
platforms. This is a necessary step to take before you can configure Machine ID
to produce credentials for your machines to use.

If a specific guide does not exist for your platform, the
[Linux guide](./platform-guides/linux.mdx) is compatible with most platforms.

## Virtual Machines

- [Linux](./platform-guides/linux.mdx): How to install and configure Machine ID on a Linux host.
- [GCP](./platform-guides/gcp.mdx): How to install and configure Machine ID on a GCP VM.
- [AWS](./platform-guides/aws.mdx): How to install and configure Machine ID on an AWS EC2 instance.
- [Azure](./platform-guides/azure.mdx): How to install and configure Machine ID on an Azure VM.

## CI/CD

- [CircleCI](./platform-guides/circleci.mdx): How to install and configure Machine ID on CircleCI.
- [GitLab](./platform-guides/gitlab.mdx): How to install and configure Machine ID on GitLab CI.
- [GitHub Actions](./platform-guides/github-actions.mdx): How to install and configure Machine ID on GitHub Actions.
- [Jenkins](./platform-guides/jenkins.mdx): How to install and configure Machine ID on Jenkins.

## Containers

- [Kubernetes](./platform-guides/kubernetes.mdx): How to install and configure Machine ID on Kubernetes.
