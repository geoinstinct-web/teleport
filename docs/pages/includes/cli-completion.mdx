<Details title="Improve the CLI experience: enable shell completion">

  Out of the box, CLI tools can take advantage of completion hints for flags and commands.
  For example, typing `tsh` and pressing `Tab` will show all available subcommands, typing `tsh --` and pressing `Tab`
  will show all available flags.

  Supported shells are bash and zsh.

  To enable completion add an additional statement to `bash_profile` (or equivalent).
  Replace `<cli-tool>` with `teleport`, `tsh`, `tctl` or `tbot`.

  Example (`.bashrc`):
  ```sh
  eval "$(<cli-tool> --completion-script-bash)"
  ```

  Example (`.zshrc`):
  ```sh
  # enable completion feature
  autoload -Uz compinit
  compinit

  eval "$(<cli-tool> --completion-script-zsh)"
  ```

  After that, reload your shell to see the changes.

</Details>
